# KeyTrainer is a modern application designed to improve reaction speed and accuracy when pressing keyboard and mouse buttons.
The program allows you to select any keys you want to train, set a custom delay, and practice in real time.

 Features

Choose any keys for training (mouse buttons included)

Displays random combinations of selected keys

Delay settings applied instantly without restarting

Minimalistic black UI with a futuristic 2025 look

Bold white font for displayed key prompts

Runs as a standalone window without a console

Global keyboard & mouse handling via pynput
